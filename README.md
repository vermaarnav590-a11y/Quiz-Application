# 🧠 Quiz Application

A command-line Quiz Application built using Python.

## Features

- Start Quiz
- Multiple Choice Questions
- Score Calculation
- Percentage Calculation
- Grade System
- Play Again Option

## Technologies Used

- Python 3

## Project Structure

```
Quiz-Application/
│
├── main.py
├── README.md
└── screenshots/
```

## How to Run

```bash
python main.py
```
